## 📌 Task 01 — Movie Genre Classification

### 🎯 Objective
Create a machine learning model to predict the genre of a movie based on its plot summary using NLP techniques.

### 📊 Dataset
- **Source:** IMDB Genre Classification Dataset
- **Records:** 54,214 movies
- **Genres:** 27 different genres
- **Top 10 Genres Used:** Drama, Documentary, Comedy, Short, Horror, Thriller, Action, Western, Reality-TV, Family

### 🔧 Technologies Used
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib, Seaborn
- TF-IDF Vectorizer
- Kaggle Notebook

### 🤖 Models Used
| Model | Type |
|---|---|
| Naive Bayes | MultinomialNB |
| Logistic Regression | Linear Classifier |
| SVM | LinearSVC |

### 📉 Graphs
- **Graph 1:** Genre Distribution Bar Chart
- **Graph 2:** Model Comparison Graph
- **Graph 3:** Confusion Matrix

## 📌 Task 03 — Customer Churn Prediction

### 🎯 Objective
Develop a model to predict customer churn for a subscription-based service using historical customer data.

### 📊 Dataset
- **Source:** Bank Customer Churn Prediction Dataset
- **Records:** 10,000 customers
- **Features:** Age, Credit Score, Balance, Geography, Gender etc.
- **Target:** Churn (1) or Not Churn (0)

### 🔧 Technologies Used
- Python, Pandas, NumPy
- Scikit-learn, Matplotlib, Seaborn
- StandardScaler, LabelEncoder
- Kaggle Notebook

### 🤖 Models Used
| Model | Type |
|---|---|
| Logistic Regression | Linear Classifier |
| Random Forest | Ensemble Method |
| Gradient Boosting | Boosting Method |

### 🔍 Key Insights
| Feature | Importance |
|---|---|
| Age | Highest (0.24) |
| EstimatedSalary | Second (0.15) |
| CreditScore | Third (0.14) |
| Balance | Fourth (0.13) |

### 📉 Graphs
- **Graph 1:** Model Comparison Graph
- **Graph 2:** Confusion Matrix
